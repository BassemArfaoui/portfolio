Weazy: The Future of E-Commerce

About Weazy

Weazy (Web + Easy) is a next-generation, intelligent e-commerce platform powered by AI. It reimagines online shopping by replacing traditional search and Browse interfaces with a chat-based system that delivers a seamless, conversational experience â€” just like ChatGPT, but made for shopping. The platform is organized into separate Frontend (React), Backend (Go), and AI/ML (FastAPI) repositories.

Technologies Used

Frontend : React : The frontend is a sleek, single-page application (SPA) built with React. It provides a smooth, chat-like user experience and is styled with Tailwind CSS for a modern and responsive interface.

Backend : Go : The backend is implemented using GoFiber, a fast and minimalist web framework for Go. It exposes RESTful endpoints to manage core business logic like user authentication, chats, and wishlists.

AI/ML : FastAPI : The AI/ML layer, built with FastAPI, serves as the brain of the platform. It interprets user queries, selects the appropriate AI tool, and interfaces with vector databases for similarity searches.

Database : PostgreSQL : All structured data, including users, chats, wishlists, and product information, is stored and managed using PostgreSQL, a powerful, open-source object-relational database system known for its reliability.

Key Features

Conversational Shopping Interface : Weazy reimagines online shopping with a chat-based system. Instead of traditional Browse, you can ask for what you want using natural language or images, just like talking to a shopping assistant.

Smart Search Options : Utilizes multiple search methods for maximum accuracy. Supports semantic Text Search using OpenAI's CLIP, Image Search based on embeddings from models like ResNet50, and Hybrid Search combining both text and images for precise results.

AI Insights & Wishlist : Save your favorite products for later with the integrated Wishlist System. You can also generate AI-powered Product Descriptions on demand to get more details about any item.

Future Enhancements : Exciting features are on the way, including DeepSearch which leverages multiple models for superior accuracy, personalized Recommendations based on user activity, and One-Click Redirection or Purchase options.

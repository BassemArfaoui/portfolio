About Tech.me

Griddy is an innovative tool designed to simplify grid-based layout challenges. Whether you're dealing with CSS Grid, Flexbox, or traditional table structures, Griddy provides a seamless way to visualize, debug, and optimize your grid layouts.

Technologies Used

    • Frontend: React - A JavaScript library designed for building complex, dynamic, and interactive user interfaces efficiently. React's component-based architecture allows for reusable and modular UI development, making it perfect for applications like Griddy, which require frequent updates and dynamic interactions, such as real-time grid visualization.
    • Backend: FastAPI - A modern, high-performance Python framework for building APIs. FastAPI leverages Python's type hints to deliver data validation and serialization out of the box. Its asynchronous capabilities ensure smooth handling of multiple requests, which is crucial for optimization tasks powered by Gurobi, such as solving the shortest path and polynomial approximation problems.
    • Desktop App: Electron.js - A framework for building cross-platform desktop applications using web technologies like JavaScript, HTML, and CSS. Electron integrates seamlessly with React for the frontend and FastAPI for backend APIs, allowing Griddy to deliver a native application experience on Windows, macOS, and Linux while maintaining a shared codebase.

Key Features

  • Interactive Grid Visualization: Users can visualize grid configurations dynamically. The React frontend renders the grid based on user inputs, showing blocked squares, start and end points, and the computed optimal paths.
  • Real-Time Updates: Integrated real-time updates let users modify grid layouts or blockages and instantly see recalculated results, enhancing the decision-making process in dynamic environments.
  • Support for Varying Grid Sizes and Weights: Griddy allows developers to experiment with grids of different sizes (up to 22x22 under a free license) and introduces weighted paths for modeling real-world traversal costs, such as terrain difficulty.
  • React and FastAPI Integration: React handles user interactions and grid rendering, while FastAPI manages computational backend tasks like shortest path calculations, polynomial fitting, and router placement optimization using Gurobi.
  • Responsive Design: Griddy is accessible on both web and desktop platforms. It adapts to varying screen sizes and devices, ensuring an optimal experience for all users, including those on mobile or low-resolution devices.
  • Robust Error Handling: The application prevents invalid configurations (e.g., blocking start/end points) and provides clear error messages when solutions are infeasible, ensuring a user-friendly experience even under complex problem constraints.
